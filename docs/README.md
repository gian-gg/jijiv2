## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Database**: PostgreSQL + Drizzle ORM
- **Auth**: Better Auth (Google OAuth)
- **State**: Zustand
- **AI**: Vercel AI SDK + OpenRouter/Gemini
